# AI Slop Detection Guide

> "AI 슬롭": AI가 학습 분포의 기본값으로 생성하는 뻔한 패턴

## 시각적 슬롭 체크리스트

### 색상
- [ ] 보라색-파란색 그라데이션 배경
- [ ] #6C63FF, #7C3AED 등 "AI 보라색"
- [ ] 밝은 그라데이션 카드 배경
- [ ] 과도한 그림자 (shadow-lg, shadow-xl)

### 레이아웃
- [ ] 3열 카드 그리드
- [ ] Hero + 3 Feature Cards + CTA 버튼
- [ ] 좌측 사이드바 + 우측 콘텐츠
- [ ] 불필요한 여백 낭비

### 컴포넌트
- [ ] shadcn/Tailwind 기본 스타일 그대로
- [ ] 둥근 카드 + 아이콘 + 제목 + 설명
- [ ] 기본 폰트 무변경
- [ ] Lucide/HeroIcons 기본 세트 그대로

### 텍스트
- [ ] "Welcome to..." 타이틀
- [ ] "Get started", "Learn more" 제네릭 CTA
- [ ] 의미없는 플레이스홀더

### 인터랙션
- [ ] hover시 scale(1.05) + shadow 증가
- [ ] 페이드인만 반복
- [ ] 스크롤 트리거 없는 정적 페이지

## 탈출 전략 (제너레이터용)

1. **색상**: 브랜드 고유 팔레트. 보라색 계열 회피
2. **레이아웃**: 비대칭, 겹침, 비표준 그리드
3. **타이포**: 2+ 폰트 조합, 크기/두께 대비
4. **인터랙션**: 마이크로 인터랙션, 커스텀 커서, 스크롤 효과
5. **원칙**: "AI가 처음 만들 것 같은 것"을 피하라
