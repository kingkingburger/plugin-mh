# How I Build Systems to STAY Focused (CS Student)

## 요약

CS 학생인 Ignant가 디지털 기기와 앱 사용을 의도적으로 통제하여 집중력을 유지하는 시스템을 소개한다. 핵심은 "쉬운 도파민"을 제공하는 앱(TikTok, YouTube, Instagram 등)에 대한 접근을 의도적으로 어렵게 만들어, 실제 인지적 작업(공부, 업무)이 상대적으로 덜 지루하게 느껴지도록 환경을 설계하는 것이다.

### 주요 포인트
1. **쉬운 도파민 앱 접근을 어렵게 만드는 시스템 구축** - 삭제, 흑백 화면 전환, 계정 로그아웃 등
2. **기기별 역할을 명확히 분리** - 폰은 소통 전용, 메인 PC는 학업 전용(YouTube 계정 로그아웃), 콘텐츠 제작 PC는 주 2일만 사용
3. **"왜" 이걸 하는지 동기를 먼저 명확히 해야 지속 가능** - 단순한 금지가 아닌, 의식적이고 의도적인 기기 사용이 핵심

## 인사이트

### 핵심 아이디어

**"환경 설계로 의지력 의존을 최소화하라"**

의지력으로 SNS를 참는 것이 아니라, 시스템(환경)을 바꿔서 쉬운 도파민에 접근하기 어렵게 만든다. 이렇게 하면 인지적으로 어려운 작업(공부, 코딩)이 상대적으로 덜 지루해진다.

구체적 시스템:
- **폰 시스템 1**: 화면을 흑백으로 전환 → SNS 콘텐츠의 시각적 자극 제거
- **폰 시스템 2**: SNS 앱 완전 삭제, WhatsApp만 남김 → 폰 = 소통 전용 기기
- **메인 PC**: 대학 계정만 로그인, YouTube 개인 계정 로그아웃 → 추천 알고리즘 차단
- **콘텐츠 PC**: 금~토만 사용 → 시간적 격리로 학업 모드 유지

### 적용 가능한 점

1. **기기 역할 분리**: 업무/학습용 기기와 엔터테인먼트 기기를 물리적으로 분리
2. **계정 전략**: 업무 PC에서 YouTube/SNS 개인 계정 로그아웃 → 추천 알고리즘 무력화
3. **시간 격리**: 콘텐츠 소비/창작을 특정 요일로 제한
4. **동기 먼저**: 시스템을 도입하기 전에 "왜 이걸 하는가"를 명확히 정의
