## 작성 가이드

### 상향 목표
- 구체적이고 측정 가능해야 한다
- 기간이 명시되어야 한다
- 원래 목표와의 차이(배율)가 명확해야 한다

### 근거
- 최소 3개, 각각 독립적인 논거
- 가능하면 실제 사례나 데이터 포함
- "가능하다"가 아니라 "왜 가능한가"를 설명

### 액션플랜
- 시간 단위별로 구분 (즉시/단기/중장기)
- 각 액션은 구체적이고 실행 가능해야 한다
- 체크포인트를 포함하여 진행 상황 추적 가능

### 리스크
- 낙관 편향을 방지하기 위해 반드시 포함
- 각 리스크에 대한 구체적 대응 방안 명시
- 발생 가능성과 영향도로 우선순위 판단

### 성공 지표
- 선행 지표: 방향이 맞는지 조기 확인
- 후행 지표: 최종 달성 여부 판단
- 체크포인트: 중간 점검 시점
